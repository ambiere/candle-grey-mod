## Dark Monochrome

>Dark monochrome colorscheme for vim <br>with a simple accent color
inspired by vesper colorscheme.

## Installation

Plugin manager: `Lazyvim`
```bash
{
    "ambiere/dark-mono",
    name = "dark-mono",
    priority = 1000,
    config = function()
      vim.cmd([[colorscheme dark-mono]])
    end,
  },
}
```
